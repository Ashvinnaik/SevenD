# Archive

Closed sprints are moved here to keep the working directories clean.

**Naming convention:** Files keep their original name (e.g., `D5-Sprint1.md`).

**When to archive:** After a sprint is closed (zero bugs) and the sprint report is filled out.

**How to archive:**
1. Move the sprint file here
2. Update the phase INDEX.md: set Archived column to `reference/archive/`
3. Update STATUS.md: move from Active to Recently Closed
