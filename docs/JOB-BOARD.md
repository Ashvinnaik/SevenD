| Status | GitHub Projects Columns | What's Here |

| **Doing** | Document, CP-4 Queue, Diagnose, CP-5 Queue, Develop, CP-6 Queue | Work in progress |
| **Done** | Deploy, CP-7 Queue, Done | Shipped and verified |
