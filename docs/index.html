A progressive 7-phase framework that gives your project a skeleton any AI coding agent can follow.

Get it on GitHub April Release notes

Most developers and builders jump straight to the code. They open Cursor, Claude Code, or Copilot, describe what they want, and let the agent run. The first few features come out fast. It feels like magic.

Then the agent invents component names that don't exist. It skips the tests. It rebuilds something you already had, and quietly breaks something you didn't ask it to touch. The velocity that felt like a superpower turns into a pile of decisions nobody made on purpose.

The problem was never the agent. It was the missing structure around it.

SevenD is that structure — seven phases that tell an AI agent what to read, what to build, and exactly when to stop and ask. Human-AI collaboration with guardrails, not vibes.

The 7 Ds

A Product loop and a Tech loop, joined at Documentation. Phase gates block premature advancement — no Design without an approved Definition, no Deployment without Diagnostics passing.

Product loop
1 · Discovery

Find what to build

Identify what to build and why before a line of code exists.

2 · Definition

Specify it

Write requirements and acceptance criteria the agent can verify against.

3 · Design

Architect it

Lock components, interfaces, and standards so the agent stops inventing.

Bridge
4 · Documentation

Sync point

Track status across every phase. One glance to know where the project stands.

Tech loop
5 · Development

Build it

Implement in time-boxed sprints against the approved design.

6 · Diagnostics

Verify it

Test for correctness and quality. Sprints close only at zero open bugs.

7 · Deployment

Ship & learn

Release, monitor, and feed lessons back into Discovery.

Quick start

Copy the templates, or let the Claude skill set it up with your real project details instead of placeholders.

Option 1 · Copy the templates

# Clone the repo git clone https://github.com/Ashvinnaik/SevenD.git # Copy the level you need cp -r SevenD/level-1/ my-project/ # prototype cp -r SevenD/level-3/ my-project/ # complex

Option 2 · Claude skill (recommended)

# Install the skill cp -r SevenD/skill/ ~/.claude/skills/SevenD/ # Start a conversation claude # Triggers when you say: # "set up my project" / "add a feature"
